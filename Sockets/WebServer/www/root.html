You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=(an integer)&num2=(an integer) -- returns the result of two numbers multiplied
  • /github?query=users/(a username)/repos -- returns the public repositories of that user
  • /madlib?noun=(a noun string)&verb=(a verb string) -- returns a small mad lib using the given words
    • Example - /madlib?noun=plane&verb=dive -- The plane decides to dive very fast.
  • /lines?min=(an integer)&max=(an integer) -- returns a string that makes three lines at random lengths based on the min and max
    • Example - /lines?min=2&max=4 -- ___|__|____|
